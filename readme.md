## Prompts Used To Make This Game

### Prompt 1
Create a modern Snake Game using HTML, CSS, and JavaScript.

Requirements:
- Classic snake movement using arrow keys
- Score counter
- Game over screen
- Restart button
- Food appears randomly
- Snake grows after eating food
- Collision detection with wall and snake body
- Modern dark UI design
- Responsive layout for desktop and mobile
- Add creative features like high score, sound effects, increasing speed, and colorful food

Generate complete working code in separate files:
1. `index.html`
2. `style.css`
3. `script.js`

### Prompt 2
Improve the Snake Game with the following requirements.

Gameplay:
- The game should not end when the snake hits any of the four walls.
- Implement screen wrapping:
  - If the snake exits from the right edge, it should appear on the left edge.
  - If it exits from the left edge, it should appear on the right edge.
  - If it exits from the top edge, it should appear at the bottom edge.
  - If it exits from the bottom edge, it should appear at the top edge.
- The game should end only when the snake collides with its own body.

### Prompt 6
Asked to apply the recommended improvements to the project.

Functionality added:
- Added a compact settings panel above the game board.
- Added difficulty modes: `Chill`, `Normal`, and `Fast`.
- Added a `Walls` toggle to switch between wraparound movement and wall-collision game over.
- Added a `Sound` toggle for the Web Audio sound effects.
- Added a `Grid` toggle to show or hide the board grid.
- Added `Space` as a keyboard shortcut to start, restart, pause, and resume.
- Improved direction handling to reduce accidental instant reversals.
- Improved mobile friendliness by allowing vertical scrolling on small screens.
- Cleaned visible game text and controls to avoid broken encoded characters.
